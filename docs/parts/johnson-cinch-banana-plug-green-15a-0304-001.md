## Wire Termination

This is a solderless banana plug — strip the wire, insert it into the barrel, and tighten the set screw:

1. Strip approximately 10mm of insulation from the wire end.
2. Unscrew the barrel of the banana plug to expose the wire clamp cavity.
3. Insert the stripped wire into the cavity.
4. Tighten the set screw firmly — the screw compresses the wire against the barrel for a gas-tight connection.
5. Reassemble the outer insulating shell.

## Color Code Convention

Banana plugs are color-coded by convention (not by regulation). Common assignments:

| Color | Typical Use |
|-------|-------------|
| Red | Positive / V+ |
| Black | Negative / Ground / Common |
| Green | Earth ground / safety ground |
| Yellow | Signal / auxiliary |
| White | Signal / auxiliary |
| Blue | Signal / auxiliary |

Green is most commonly used for earth/ground connections in test equipment and bench power supplies.

## Compatibility

Standard 4mm banana plugs mate with:
- **Banana jacks** (panel-mount or PCB) — the universal test equipment connector
- **Binding posts** — common on bench power supplies and speaker terminals
- **Multimeter inputs** — standard probe connectors
- **BNC-to-banana adapters** — for oscilloscope connections

The 15A / 3.5kV rating makes this suitable for most bench test applications. It exceeds the ratings of typical multimeters and bench supplies.
