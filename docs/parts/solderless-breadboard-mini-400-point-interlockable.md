# Solderless Breadboard Mini — 400-Point Interlockable

A half-size prototyping breadboard with 400 tie-points. Same 0.1" pitch as the full-size MB-102 and interlocks with it via side tabs — use it to extend your prototyping area or as a compact standalone board for small circuits. Fits DIP ICs and standard through-hole components.

## Specifications

| Spec | Value |
|------|-------|
| Tie Points | 400 |
| Center Rows | ~30 rows x 2 sides |
| Columns per Side | 5 (a-e, f-j) |
| Power Rails | 2 (1 per side, +/-) |
| DIP IC Gap | 0.3" (7.62mm) |
| Interlocking | Side tabs — locks to other mini or full-size boards |
| Dimensions | ~85 x 55 x 10mm |

## Usage Notes

Best used as a satellite board next to a full-size breadboard — interlock it on the side and run jumper wires across. Good for isolating a sub-circuit (sensor conditioning, voltage regulation, transistor driver) from the main logic on the big board.
