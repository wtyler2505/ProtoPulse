# index

Welcome to your electronics inventory system. Every component you own is documented, cross-referenced, and linked.

## Categories

- [[microcontrollers]] -- Arduino, ESP, Raspberry Pi, development boards
- [[sensors]] -- distance, motion, temperature, light, sound, environmental
- [[actuators]] -- motors, servos, steppers, relays
- [[displays]] -- LCD, LED, OLED, 7-segment, matrix
- [[power]] -- batteries, regulators, power modules, breadboard supplies
- [[shields]] -- Arduino shields, breakout boards, expansion modules
- [[communication]] -- WiFi, Bluetooth, RFID, IR, Ethernet modules
- [[passives]] -- resistors, capacitors, discrete semiconductors, LEDs
- [[input-devices]] -- buttons, potentiometers, joysticks, encoders, keypads

## Reference

- [[wiring-guides]] -- connection recipes between components
- [[compatibility]] -- voltage, interface, and I2C address matrices
- [[unidentified-parts]] -- components that need identification

## Getting Started

1. Read self/identity.md to understand how this system works

3. Cross-reference it with /arscontexta:cross-reference
4. Check system health with /arscontexta:health
