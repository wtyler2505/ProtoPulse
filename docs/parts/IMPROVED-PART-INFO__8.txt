Function: The primary user interface for initiating an emergency stop.

Mechanism: It uses a Normally Closed (NC) contact. When the button is pressed, the NC contact opens, interrupting the control circuit.

Coil Circuit: The contactor's coil is part of the control circuit. When energized, it closes the main power contacts.

Fail-Safe Principle

The core safety principle relies on using Normally Closed (NC) contacts for the control circuit. This design ensures that any failure in the control circuit—not just pressing the button—results in a safe state (power cutoff).

Standard Operation: When the system is on, the NC contact in the e-stop button is closed, sending power to the contactor coil. The coil remains energized, keeping the power contacts closed, and power flows to the load.

Emergency Stop Activation: Pressing the e-stop button opens the NC contact, de-energizing the contactor coil. The power contacts spring open, cutting off high-voltage power.

Fail-Safe Condition (Wire Break): If the wire connecting the e-stop button to the contactor coil breaks, power to the coil is lost. This is functionally identical to pressing the button, causing the high-power contacts to open and cut power.

The circuit path goes from the buck converter output through the e-stop button's NC contact (terminals 11 and 12).

The output of the NC contact connects directly to the positive terminal of the contactor coil.

The negative terminal of the contactor coil returns to the buck converter ground.

2. Power Circuit (36V DC, 100A+):

The high-voltage source (battery via a main power switch/fuse) connects to the "Main IN" terminal of the contactor.

The "Main OUT" terminal of the contactor provides the protected power to the rest of the system (e.g., power distribution board).

This circuit path is only complete when the contactor's coil is energized.
