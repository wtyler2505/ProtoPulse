# Solderless Breadboard Full-Size — MB-102 830-Point

The standard prototyping breadboard. 830 tie-points in a 0.1" grid — 630 component holes in the center (63 rows x 5 columns x 2 sides) plus 4 power rail strips along the edges. Accepts DIP ICs, through-hole components, and 22AWG jumper wire. The Elegoo power module (in inventory) plugs directly into the power rail end.

## Specifications

| Spec | Value |
|------|-------|
| Tie Points | 830 |
| Center Rows | 63 rows x 2 sides |
| Columns per Side | 5 (a-e, f-j) |
| Power Rails | 4 (2 per side, +/-) |
| DIP IC Gap | 0.3" (7.62mm) |
| Interlocking | Side tabs for joining multiple boards |
| Dimensions | ~165 x 55 x 10mm |
