## Identification Workflow

1. Document physical markings, dimensions, pin count
2. Search for part numbers, manufacturer logos
3. Test basic electrical characteristics (voltage, resistance, behavior)
4. Match against known component databases
5. Update status from "unidentified" to "needs-test" or "verified"
