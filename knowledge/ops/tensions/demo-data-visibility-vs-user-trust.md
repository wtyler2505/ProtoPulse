DEMO badges made the mock nature of pricing data honest, but users still want real quotes. Building real supplier integrations requires OAuth flows, API key management, response caching, rate limiting, and currency conversion — significant infrastructure for each supplier. The tension: ship more demo-badged features to show capability, or block feature development until the integration infrastructure is built?
