ProtoPulse's wave model optimizes for rapid vertical delivery. Integration is consistently deferred because it spans multiple agent tasks and file ownerships. Wave 37 was entirely remediation (24 unintegrated modules). The question is whether to accept this cadence (implement → accumulate → remediate) or restructure waves to include integration tasks alongside feature work, which would slow individual waves but reduce remediation waves.
