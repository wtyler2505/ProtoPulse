# nineteen mastery skills are the deepest knowledge layer

The global skills directory contains 19 mastery-level skills: browser-use-mastery, chromedevtools-mastery, claude-code-hooks-mastery (deprecated), claude-md-mastery, clearthought-mastery, cli-research-mastery, cli-tools-mastery, codex-mastery, computer-vision-mastery, debugging-mastery, fetlife-mastery, hooks-mastery, interactive-builder-mastery, magic-mastery, mcp-server-mastery, slash-commands-mastery, tdd-mastery, testing-mastery, and verification-mastery.

Mastery skills are structurally different from regular skills. Where a regular skill encodes a workflow (do X, then Y, then Z), a mastery skill encodes expertise (here's everything you need to know about X). They function as on-demand knowledge bases that Claude loads when deep domain context is needed.

Several mastery skills subsume and deprecate older skills: debugging-mastery absorbed systematic-debugging, no-shortcuts-debugging, and root-cause-tracing. Testing-mastery absorbed complete-testing and testing-anti-patterns. Verification-mastery absorbed thorough-verification and verification-before-completion. Hooks-mastery absorbed claude-code-hooks and claude-code-hooks-mastery. This consolidation pattern suggests mastery skills are the evolutionary endpoint of skill specialization.
