ProtoPulse's wave model ships features as complete vertical slices — from schema to storage to route to hook to UI. This enables rapid delivery (79 waves, 197 done items) but creates integration debt because cross-domain wiring is deferred. Wave 37 was entirely remediation, wiring 24 unintegrated library modules into the UI. The pattern suggests integration waves should be scheduled proactively, not reactively.
