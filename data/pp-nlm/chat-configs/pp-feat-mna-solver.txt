DOMAIN VOCABULARY (use these — don't substitute generic terms)
- **MNA matrix** — the modified-nodal-analysis matrix [G][V] = [I]. G = conductance, V = node voltages, I = current sources.
- **Stamp** — the contribution of a single component to the MNA matrix. Each component "stamps" its conductance + currents into the matrix.
- **DC operating point (DCOP)** — the steady-state solution at t=0 with all reactive elements treated as their DC equivalents (capacitor = open, inductor = short).
- **Newton-Raphson iteration** — the iterative solver for nonlinear DC. Convergence is the key concern.
- **Gmin stepping** — a continuation technique: add a tiny conductance to ground from every node to aid convergence, then ramp it down.
- **Source stepping** — ramp voltage/current sources from 0 to their target value to aid convergence.
- **Damped Newton** — limit step size in NR to avoid overshoot.
- **Sparse matrix** — most circuit MNA matrices are sparse (tens of % nonzero); KLU or LU-with-permutation is the canonical solver.
- **Transient analysis** — time-domain integration (Trapezoidal default; Gear's for stiff systems).
- **Virtual multimeter probe** — UI affordance for measuring node voltage / branch current at a point in the simulated circuit.
- **SPICE netlist** — the canonical text format for circuits (R1 N1 N2 1k, V1 N3 0 5, .DC, .TRAN, etc.). The parser extracts these into the solver's internal representation.
