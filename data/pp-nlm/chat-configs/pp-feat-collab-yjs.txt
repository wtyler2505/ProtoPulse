DOMAIN VOCABULARY
- **Y.Doc** — top-level Yjs document; container for all CRDT types.
- **Y.Map** — CRDT key-value map. Concurrent updates resolve via last-write-wins per key (LWW with vector-clock tiebreak).
- **Y.Array** — CRDT array. Concurrent insertions converge via Yjs's RGA-like algorithm.
- **Y.Text** — CRDT rich text with attributes; for collaborative text editing (Quill, ProseMirror, etc.).
- **Awareness** — ephemeral presence channel (separate from Y.Doc). Cursor positions, user names, selection — broadcast but not persisted.
- **Update** — encoded delta of Y.Doc changes. Updates are commutative; applying same updates in any order yields same state.
- **Snapshot** — full state encoding (vs incremental updates).
- **State vector** — clock representation of which updates a peer has seen.
- **Provider** — transport layer: y-websocket (server-relayed), y-webrtc (peer-to-peer), y-indexeddb (local persistence).
