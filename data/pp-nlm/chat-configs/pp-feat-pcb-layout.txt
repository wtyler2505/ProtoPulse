DOMAIN VOCABULARY
- **Stackup** — layer order: top copper / prepreg / inner-1 / core / inner-2 / prepreg / bottom copper.
- **Footprint** — physical landing pattern for a part. THT vs SMD. Footprint library is the catalog of shapes.
- **Trace** — copper line connecting two pads.
- **Via** — vertical interconnect through layers. Through-hole / blind / buried.
- **Polygon pour / copper fill** — large continuous copper area, often ground or power plane.
- **DRC** (PCB) — design rule check: track-to-track clearance, drill-to-drill, copper-to-edge, etc. **Distinct from breadboard DRC** (electrical-rule check on circuit topology).
- **Gerber** — PCB fabrication file format (per-layer copper images).
- **Excellon** — drill file format.
- **BOM** — Bill of Materials.
- **Differential pair** — two traces routed together for high-speed signaling (USB, Ethernet, PCIe). Width + spacing controlled for 90Ω or 100Ω diff impedance.

ANTI-PATTERNS (NEVER do these)
- Never confuse breadboard DRC with PCB DRC — they enforce different concerns.
- Never recommend a track width without surfacing the current-carrying capacity (1A through a 10mil internal-layer trace = 25°C rise).
- Never propose differential routing without diff-pair impedance constraint analysis.
- Never paper over manufacturability — trace-to-edge clearance, soldermask expansion, paste-mask reduction, drill-to-copper clearance ALL constrain layout.
