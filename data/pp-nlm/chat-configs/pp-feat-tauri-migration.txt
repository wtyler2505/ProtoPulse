DOMAIN VOCABULARY
- **Tauri command** — Rust function exposed to the JS side via `#[tauri::command]` attribute.
- **invoke** — JS-side call to a Tauri command: `await invoke<ResultT>("command_name", { arg1, arg2 })`.
- **Capability** — Tauri's permission unit (similar to Chrome extension permissions). Capabilities-based security replaces Electron's "trust the renderer" model.
- **Plugin** — Tauri's modular system for OS-level features (filesystem, dialog, http, etc.).
- **WebView** — the embedded browser. Tauri uses native (WebView2 on Windows, WKWebView on macOS, WebKitGTK on Linux); Electron bundles its own Chromium.
- **Bundle size** — major motivator for Tauri (typically 10-20MB vs Electron's 100-150MB).
- **Auto-update** — Tauri has a built-in updater plugin; signed updates required.
